# 💪 Fitness Website

A responsive and modern Fitness Website built using **HTML, CSS, JavaScript, Bootstrap, and jQuery**. This project is designed for gyms, fitness centers, and personal trainers to showcase their services, pricing, trainers, gallery, and contact information.

📌 Features

* Responsive design for all devices
* Modern and clean user interface
* Home page with hero section
* About page
* Fitness courses section
* Membership pricing plans
* Image gallery
* Blog pages
* Contact page with contact form and Google Maps
* Smooth navigation and animation

🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap
* JavaScript
* jQuery
* Font Awesome
* Google Maps API
